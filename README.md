# AI Code Reviewer

AI-powered Code Reviewer built using React and OpenRouter API. This application analyzes code and provides bug detection, optimization suggestions, best practices, and improved code recommendations using Artificial Intelligence.

## Features

* AI Code Review
* Bug Detection
* Optimization Suggestions
* Best Practices Analysis
* Improved Code Recommendations
* Modern Responsive UI
* Real-time AI Responses

## Tech Stack

* React.js
* JavaScript
* Axios
* OpenRouter API
* CSS

## Installation

Clone the repository:

```bash id="b5w8fr"
git clone https://github.com/yourusername/AI-Code-Reviewer.git
```

Go to project folder:

```bash id="m2n7xq"
cd AI-Code-Reviewer
```

Install dependencies:

```bash id="v4t9zl"
npm install
```

Create `.env` file:

```env id="k5m1vk"
VITE_OPENROUTER_API_KEY=your_api_key
```

Run the project:

```bash id="p8w3fr"
npm run dev
```
